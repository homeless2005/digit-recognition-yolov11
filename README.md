# YOLO 数字识别

基于 YOLO11 的手写/印刷体数字识别项目，支持 0-9 共 10 个数字类别的检测与分类。

## 安装

1. 确保已安装 Python 3.8 或更高版本
2. 安装依赖包：ultralytics、opencv-python、numpy
3. 如需 GPU 加速，请安装 CUDA 版 PyTorch

## 项目说明

### 数据集

采用 YOLO 标注格式的 0-9 数字图像数据集，已按训练集、验证集、测试集划分。数据集配置位于 `data.yaml`。

### 训练

运行 `yolo_train.py` 即可启动训练。脚本以 YOLO11n 为基础模型，输入尺寸 512×512，训练 100 轮。内置了丰富的数据增强策略（Mosaic、MixUp、CopyPaste、随机翻转、HSV 变换、随机擦除等）以及 Dropout 和 Weight Decay 正则化，同时配有早停机制防止过拟合。所有超参数均可在脚本顶部直接调整。

### 本地视频检测

将需要检测的视频文件放入 `video_dir/` 文件夹，运行 `video2prt.py` 即可逐视频播放检测结果。支持的视频格式包括 MP4、AVI、MOV、MKV、FLV。播放过程中可按 Q 退出程序，按 N 或 Esc 跳过当前视频。

### 手机摄像头实时检测

通过手机 IP 摄像头 APP 将手机画面实时传输到电脑进行检测：

1. 手机下载 IP 摄像头类 APP（如 IP Webcam）
2. 确保手机和电脑连接同一 WiFi
3. 将手机 APP 显示的流地址填入 `ip_webcam2prt.py` 的 `PHONE_CAMERA_URL` 变量
4. 运行 `ip_webcam2prt.py` 开始实时检测，按 Q 退出

### 可调参数

置信度阈值、字体大小、检测框粗细、显示缩放比例等参数均在各自脚本顶部的配置区域中，可根据实际需求直接修改。

## 模型

- **基础模型**：YOLO11n — YOLO11 系列中最轻量的模型，适合快速训练和部署
- **训练产出**：best.pt — 在当前数据集上训练得到的最佳权重

本项目仅供学习和研究使用。
